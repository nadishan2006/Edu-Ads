## 🧠 Features

- 🔔 **Real-Time News Updates**
- 💬 **Stay Updated**: Email subscription feature
- 📈 **User-Friendly Interface**
- ⚡ **Blazing Fast** with React + Vite
- 🌐 **SEO Optimized**
- 🌙 **Dark Mode Ready**
